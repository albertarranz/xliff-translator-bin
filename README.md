# XLIFF Translator

A desktop application for translating XLIFF (XML Localization Interchange File Format) files across multiple languages.

## Features

- Open and edit XLIFF translation files
- Translate content between multiple languages
- Visual interface built with Avalonia UI
- Cross-platform support (Linux and Windows)

## Downloads

### Linux (x64)

Download the installer from the [Releases](https://github.com/albertarranz/xliff-translator-bin/releases) page.

**Install:**
```bash
sudo bash xliff-translator-installer.sh
```

**Uninstall:**
```bash
sudo bash xliff-translator-installer.sh uninstall
```

### Windows (x64)

Download and run `xliff-translator-windows-setup.exe` from the [Releases](https://github.com/albertarranz/xliff-translator-bin/releases) page.

## Building from Source

This repository contains pre-built binaries. For the source code, see [XliffTranslator](https://github.com/albertarranz/XliffTranslator).
